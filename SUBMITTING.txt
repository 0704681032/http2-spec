teps for submitting a new version of draft-ietf-httpbis-http2

draft number: NN

1) cd draft-ietf-httpbis-http2/latest

2) Check for "None yet" in the "Changes" section and change it to "None".

3) cd .. ; svn cp latest NN

4) cd NN

5) Change @docName attribute from -latest to -NN

6) Set @month and @day on the date element to reflect the submission day.

7) make

8) svn cp draft-ietf-httpbis-http2.xml draft-ietf-httpbis-http2-NN.xml ; svn rm --force draft-ietf-httpbis-http2.xml

9)  mv draft-ietf-httpbis-http2.txt draft-ietf-httpbis-http2-NN.txt ; svn add draft-ietf-httpbis-http2-NN.tx ; tsvn ps svn:eol-style native draft*txt

10) cd ../latest

11) Add "Since draft-ietf-httpbis-http2-...-NN" subsection, and update the link from the boilerplate.

12) make

13) # review the diffs, then commit

14) # don't forget to submit the drafts
